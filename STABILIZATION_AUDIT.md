## 🛠 Fixes & Enhancements Implemented

### 1. Digital ID Card & Global UI State

- **Feature:** Integrated the Digital ID Card as a global modal dialog.
- **Action:** Created `DigitalIDDialog.tsx` and `useUIStore` (Zustand) to manage its state globally.
- **Optimization:** Intercepted Sidebar clicks to open the modal instead of navigating to a separate page, providing a seamless mobile-first experience.
- **Enhancement:** Upgraded the `DigitalCard` component to display active gate pass details (departure/return times, destination, and status) with premium animations.

### 2. Authentication & Permission Resilience

- **Issue:** The application was incorrectly logging out users on generic `403 Forbidden` errors during page refresh (bootstrap phase).
- **Fix:** Updated `App.tsx` bootstrap logic to only trigger logout on `401 Unauthorized` status. Generic `403` errors are now handled by the API interceptor or RoleProtectedRoute, allowing for more specific error handling (e.g., "College is temporarily disabled").

### 3. Profile Image Upload Optimization

- **Fix:** Validated image size (up to 5MB) and formats (JPG, JPEG, PNG) on the frontend to match backend constraints.
- **Fix:** Secured the `bulk_upload` process in the backend to process users individually within isolated transitions, ensuring that a single malformed row doesn't roll back the entire batch.

### 4. Mobile Layout & UI Glitches

- **Sidebar Fix:** Increased the bottom padding of the navigation categories in `Sidebar.tsx` to prevent the last menu items and the logout button from being obstructed by mobile navigation bars or safe area insets.
- **Workflow Consistency:** Fixed validation logic in `LeavesPage.tsx` where optional fields (Parent Contact, Destination) were incorrectly treated as mandatory for button enabling.

### 5. Backend Refactoring (Resilience)

- **User Bulk Upload:** Refactored the `bulk_upload` endpoint in `auth/views.py`. It now performs individual atomic saves per user, providing detailed error reporting per row while maintaining data integrity.
